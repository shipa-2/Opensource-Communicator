# Changelog

Все заметные изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [0.2.0] — 2026-07-09

### Добавлено

- **Запись разговоров** — во время звонка пишется WAV PCM 16-bit mono 48 kHz, после завершения конвертируется в MP3 (`ffmpeg`/`lame`); настройки в «Настройки → Запись разговора»: вкл/выкл, папка, шаблон имени файла (`%d`, `%m`, `%y`, `%h`, `%min`, `%s`, `%name`), опция двух дорожек (смешанная + оператор + абонент).
- **Вкладка «История»** — фильтр периода («Показать за»), направление (все / входящие / неотвеченные / исходящие), поиск, вкладки области (все / внутренние / внешние), время ожидания в строках; клик по записи открывает заметку без кнопки «Позвонить».
- **Конференция** — двухсписочный диалог: собеседники и слушатели, перенос стрелками; роли `_manager` / `_caller` в JSON.
- **Контакты** — меню «Добавить»: контакт / импорт CSV и vCard; ПКМ по 📞 — «Позвонить по:»; двойной клик — заметка с кнопкой «Позвонить»; ПКМ — удаление (локальные) и экспорт vCard/CSV.
- **Меню «Вид»** — показ/скрытие кнопки чата (сохраняется в настройках).
- **Вход** — пустые логин и пароль автоматически подставляют демо `demo` / `demo`; «Запомнить меня» с автологином и списком сохранённых аккаунтов (demo не сохраняется).
- **Слепой перевод** — диалог выбора цели, запись в истории, окно звонка закрывается после перевода.

### Исправлено

- **Звук в звонках** — устранены тишина и задержка аудио на Megafon PBX (отложенная WebRTC-negotiation, RTP timestamps, `disableAutoNegotiation`).
- **Конференция** — корректные флаги `muted` для слушателей.
- **Помощь на Windows** — картинка справки переведена в PNG (JPEG не открывался без плагина `qjpeg` в portable-сборке).
- **Чат** — исходящие сообщения больше не дублируются; история по времени (новые снизу) с метками `ЧЧ:ММ` / `29 янв. ЧЧ:ММ` и коротким `RealName`.
- **Запись** — смешанная дорожка больше не растягивается; таймер и запись стартуют с первого слышимого Opus от собеседника.

### Изменено

- **Шапка** — «Настройки» и «Помощь» вынесены в строку меню слева (без троеточий).
- **Подвал** — «Добавить» и «Конференция» без лишних символов; селектор периода истории — ссылка и нативное меню.
- **CI** — сборки Linux и Windows (Release + Debug); при теге `v*` артефакты прикрепляются к GitHub Release.

## [0.1.0] — 2026-03

### Добавлено

- Первый публичный релиз: WebSocket-протокол ITooLabs, контакты, presence, чат (IM + SMS), звонки WebRTC/Opus, удержание, конференция, демо-режим, нативный Qt 6 UI.
- Сборка Windows (portable ZIP) и документация протокола (`PROTOCOL.md`).
- Пакет AUR `opensource-communicator-git`.
