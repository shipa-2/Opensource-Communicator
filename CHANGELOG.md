# Changelog

Все заметные изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [0.4.0] — 2026-07-18

### Добавлено

- **Сервер `communicator-server`** — самостоятельный сигнализационный сервер в каталоге `server/` (WebSocket, IM, presence, звонки, hold music, PostgreSQL). **Не входит в GitHub Release** — только исходники; сборка локально, см. [server/README.md](server/README.md).
- **OSC: обмен темой и аватаркой** — «Поделиться темой» / «Поделиться аватаркой» в настройках аккаунта; IM `**fnm=…**`, подтверждение `Themeapplied!`; обои 390×620 с двумя ползунками затемнения.
- **Вложения в чате** — кнопка «+», IM `**fnm=…;enc=b64;…**` (до 96 КБ), уведомление «*Имя* отправил файл» с сохранением по клику.
- **Кликабельные URL** в тексте сообщений чата.
- **OSC UX** — кольцо присутствия у OSC-пиров, волна при обнаружении; строка «Мой аккаунт (Я)» всегда с OSC-кольцом; в демо через ~5 с **Администратор** присылает `Hello world.txt`.
- **OSC-only IM: Openping!, аватарка, тема** — обнаружение клиентов OpenSource Communicator, обмен фото аватарки (140×140 PNG) и темой; см. [PROTOCOL.md](PROTOCOL.md#расширения-opensource-communicator-im).
- **Обои и тема** — фон главного окна с двумя ползунками затемнения.
- **DTMF в окне звонка** — кнопка «Тон», in-band DTMF через Opus, ввод с клавиатуры (`0–9`, `*`, `#`).
- **Список истории звонков** — отдельный виджет строки с hover/выделением, ПКМ, двойной щелчок → заметка с «Позвонить» и «Закрыть».
- **Кнопка видеозвонка** — при `videoEnabled` из `getcommunicatorsettings` (сервер с `--allowvideo`): 📹 у контакта, «Видеозвонок» в ПКМ; скрытие через «Вид → Кнопка видеозвонка» (сам видеопоток в клиенте пока не реализован).
- **Второй экземпляр клиента** — флаг `--newinstance` (отдельный single-instance и tel:-обработчик).
- **Подключение к своему серверу** — поле «Порт сервера» в расширенном входе; в debug-сборке — «Игнорировать небезопасный TLS» (`ws://`, self-signed `wss://`).
- **Debug: рассылка цвета аватарки** — кнопка «Рассылка цвета» в «Настройки → Аккаунт» (только debug-сборка).

### Изменено

- **Кейпад «Набрать вручную»** — удержание **0** (1 с) вводит `+`; пауза перед заливкой ⌫ — 0.25 с.
- **Рассылка цвета аватарки** — отдельным IM каждому контакту домена, не через `__broadcast__`.
- **Присутствие «Говорит по телефону»** — локальный UI; клиент не шлёт `SetPresence(in-call)` на PBX, где сервер отклоняет (971).
- **Меню «Вид»** — пункт «Кнопка звонка» (ранее «Кнопка позвонить»); добавлен «Кнопка видеозвонка».
- **Подсказки контакта** — «Звонок» вместо «Позвонить» (кнопка и ПКМ).

### Исправлено

- **Удержание** — ответ `AcceptUpdate` на `updated` (renegotiation SDP); статус «На удержании» / «Удержание»; кнопка «Удержание» неактивна, когда вас держат на hold; при локальном hold исходящий голос не передаётся.
- **MPRIS** — `Play` только для плееров, которые играли до паузы.
- **Окно звонка** — аватар без зазора; VAD-кольцо поверх фото.
- **Аватарки в контактах** — фото без лишнего кольца цвета; OSC-кольцо снаружи.
- **Чат** — Enter не открывает выбор файла при фокусе на «+»; ссылка «поделился темой» не «липнет» ко всем сообщениям; имена файлов с пробелами; **IM между двумя клиентами на localhost** — нормализация `localhost` / `127.0.0.1` в ключе пира.
- **Настройки** — «Поделиться…» без segfault/hang (модальные диалоги); кнопки неактивны без OSC-пиров, подсказка почему.
- **Выбор сетевой карты для WebRTC** — в «Настройки → Аккаунт».
- **Краш debug при звонке** — `qAbs(INT16_MIN)` в расчёте уровня Opus.
- **ICE WARN после сброса** — PeerConnection закрывается сразу в teardown.
- **Краш debug (ThemeHelper)** — `Qt::UniqueConnection` с lambda.
- **Входящий без аудио / «Соединение…»** — ICE после «Принять»; bind на выбранный интерфейс.
- **Входящий и Esc** — окно не забирает фокус; Esc не сбрасывает вызов.
- **Дублирующий SetPresence после звонка** — убран лишний `SetPresence(online)`.
- **Звук уведомления о сообщении** — корректная остановка воспроизведения.

## [0.3.0] — 2026-07-09

### Добавлено

- **Контекстное меню контактов** — ПКМ по контакту показывает: Позвонить (с подменю если несколько номеров), Сообщение, Заметка, Удалить, Экспортировать.
- **Копирование номера своего контакта** — ПКМ по своему контакту предлагает скопировать личный или рабочий номер в буфер обмена.
- **Меню «Вид» — скрытие кнопки «Позвонить»** — новый пункт в подменю «Вид», сохраняется в настройках.
- **Настройки с вкладками** — диалог «Настройки» разделён на три вкладки: Звук, Аккаунт, Запись.
- **Настройки аккаунта** — аватарка с кнопками загрузки фото/выбора цвета/удаления фото (справа от аватарки), отображаемое имя с кнопкой «Сбросить», кнопка «Выйти из аккаунта».
- **Настройки записи** — объединённая дорожка (опция появляется при включении двух дорожек), динамический превью имён файлов.
- **Передача цвета аватарки** — при входе приложение отправляет цвет аватарки в формате `**#RRGGBB**`; полученные цвета применяются к аватаркам контактов. Сообщения-рекламы не отображаются в чате.
- **Интерфейс звонка** — переработан по вайрфрейму: имя/номер сверху, крупный круглый аватар (140×140) с цветом контакта, статус и таймер по центру, заметки всегда видны, кнопки «Перевод», «Сброс», «Удержание» с текстовыми подписями.
- **Подсветка голоса при звонке** — border аватарки подсвечивается акцентным цветом когда собеседник говорит (автокалибровка порога по шуму).
- **Drag-and-drop по всему окну** — перетаскивание .vcf/.csv/.txt файлов работает на всей площади окна, включая список контактов.
- **Боковая панель настроек удалена** — все настройки в диалоге с вкладками.

### Изменено

- **Кнопка «Сброс»** — акцентный цвет фона.
- **Фон окна звонка** — отличается от главного окна (`palette(window)`).
- **Аватар в шапке** — некликабельный (без меню настроек).
- **Директория записи** — переименована из «Папка», поле расширено.
- **Чекбокс «объединённая дорожка»** — всегда в layouts (серый когда недоступен), не сдвигает поле «Директория».

### Исправлено

- **Краш при звонке (debug)** — ASSERT в `qAbs` на сэмпле −32768 при расчёте уровня удалённого аудио.
- **ICE после сброса** — явное закрытие PeerConnection при завершении звонка.
- **Краш debug (ThemeHelper)** — connect с `Qt::UniqueConnection` только через member slot, не lambda.
- **Входящий звонок без аудио** — ICE и AcceptCall только после ответа; выбор сетевой карты в настройках аккаунта (WebRTC bind).
- **Кнопка «Перевод»** — больше не отображается как зажатая по умолчанию.
- **Drag-and-drop** — корректная обработка DragMove для отображения курсора переноса.

## [0.2.0] — 2026-07-09

### Добавлено

- **Запись разговоров** — во время звонка пишется WAV PCM 16-bit mono 48 kHz, после завершения конвертируется в MP3 (`ffmpeg`/`lame`); настройки в «Настройки → Запись разговора»: вкл/выкл, папка, шаблон имени файла (`%d`, `%m`, `%y`, `%h`, `%min`, `%s`, `%name`), опция двух дорожек (смешанная + оператор + абонент).
- **Вкладка «История»** — фильтр периода («Показать за»), направление (все / входящие / неотвеченные / исходящие), поиск, вкладки области (все / внутренние / внешние), время ожидания в строках; клик по записи открывает заметку без кнопки «Позвонить».
- **Конференция** — двухсписочный диалог: собеседники и слушатели, перенос стрелками; роли `_manager` / `_caller` в JSON.
- **Контакты** — меню «Добавить»: контакт / импорт CSV и vCard; ПКМ по 📞 — «Позвонить по:»; двойной клик — заметка с кнопкой «Позвонить»; ПКМ — удаление (локальные) и экспорт vCard/CSV.
- **Меню «Вид»** — показ/скрытие кнопки чата (сохраняется в настройках).
- **Вход** — пустые логин и пароль автоматически подставляют демо `demo` / `demo`; «Запомнить меня» с автологином и списком сохранённых аккаунтов (demo не сохраняется).
- **Слепой перевод** — диалог выбора цели, запись в истории, окно звонка закрывается после перевода.

### Исправлено

- **Звук в звонках** — устранены тишина и задержка аудио на Megafon PBX (отложенная WebRTC-negotiation, RTP timestamps, `disableAutoNegotiation`).
- **Конференция** — корректные флаги `muted` для слушателей.
- **Помощь на Windows** — картинка справки переведена в PNG (JPEG не открывался без плагина `qjpeg` в portable-сборке).
- **Чат** — исходящие сообщения больше не дублируются; история по времени (новые снизу) с метками `ЧЧ:ММ` / `29 янв. ЧЧ:ММ` и коротким `RealName`.
- **Запись** — смешанная дорожка больше не растягивается; таймер и запись стартуют с первого слышимого Opus от собеседника.

### Изменено

- **Шапка** — «Настройки» и «Помощь» вынесены в строку меню слева (без троеточий).
- **Подвал** — «Добавить» и «Конференция» без лишних символов; селектор периода истории — ссылка и нативное меню.
- **CI** — сборки Linux и Windows (Release + Debug); после успешной сборки `main` Release-артефакты публикуются в GitHub Release с тегом **ДДММГГГГ**.
